# lamda表达式

语法很简单，不多说了

唯一重要的是捕获列表

```cpp
int c=1;
int d=2;
auto add =[c,d](int a,int b)->int{
return a+b+c+d;
};
d=3;
int res=add(3,4);
```

此时不会因为后面修改了d的值在调用add的时候就改变了a+b+c+d的值，捕获是发生在写捕获列表的时候的

### 捕获列表的四个重要用法

```cpp
auto func=[&](){}// &代表以引用的方式捕获所有外部变量‘
auto func=[=](){}//以传值的方式捕获所有外部变量
auto func=[](){}//不捕获任何外部变量
auto func=[this](){} //这个必须要在类函数中使用，这代表函数内部可以直接访问本类的私有成员函数和变量
```
